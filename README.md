# symfony4-docker

## Directory structure:
----------------------------------------------------------------------------------------------------------
- project-root/
    - docker/
        - mysql/
        - nginx/
            - default.conf
        - php/
            - xdebug.ini
            - Dockerfile
    - docker-compose.yml
----------------------------------------------------------------------------------------------------------
